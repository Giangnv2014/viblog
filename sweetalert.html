Sweet Alert Plugin

Posted on 2day ago by Nguyen Van Giang

Sweet Alert là một jquery plugin dùng để thay thế cho hộp thoai alert mặc định của javascript. Nó sẽ giúp bạn tô điểm cho website của bạn thêm lung linh hơn.

Giả sử website của bạn cũng đẹp như Viblo của Framgia chúng tôi và bạn sử dụng alert mặc định của javascript:

So với việc bạn sử dụng sweet alert:

Đến đây bạn cũng đã hiểu lý do tại sao nên chọn sweet alert cho ứng dụng web của mình. Nó không chỉ có ưu điểm về giao diện vượt trội so với hộp thoại alert mặc định, mà cách cài đặt và như sử dụng cũng rất đơn giản:

Cài đặt

  • Sử dụng Bower 
    $ bower install sweetalert

  • Cài đặt thông thường

    Download sweetalert tại đây, sau đó include vào website của bạn:

    <script src="lib/sweet-alert.min.js"></script>
<link rel="stylesheet" type="text/css" href="lib/sweet-alert.css">

Sử dụng

Gọi phương thức swal


            swal({
               // Tham số : giá trị
            });
Tham số Giá trị mặc định Mô tả
title null (required) Tiêu đề hộp thoại alert
text null Mô tả hộp thoại alert
type null Loại alert: SweetAlert cung cấp 4 loại đã được xây dựng sẵn: "warning", "error", "success" và "info".
allowOutsideClick false Cho phép tắt bằng cách bấm ra ngoài hội thoại
showCancelButton false Hiển thị nút Cancel, dùng để đóng hộp thoại
confirmButtonText "OK" Thay đổi chữ cho nút Confirm. Nếu showCancelButton được đặt là true, thì sẽ đặt tự đồng là "Confirm" thay vì "OK".
confirmButtonColor "#AEDEF4" Thay đổi mầu nền của nút "Confirm" (giá trị HEX).
cancelButtonText "Cancel" Thay đổi chữ của nút "Cancel"
closeOnConfirm true Đặt false nếu bạn muốn hộp thoại tiếp tục mở khi người dùng bấm nút "Confirm". Chức năng này hữu dụng khi hàm callback gắn với nút Confirm gọi đến một hộp thoại SweetAlert khác.
closeOnCancel true Đặt false nếu bạn muốn hộp thoại tiếp tục mở khi người dùng bấm nút "Cancel". Chức năng này hữu dụng khi hàm callback gắn với nút Cancel gọi đến một hộp thoại SweetAlert khác.
imageUrl null Đường dẫn gắn hình ảnh iconcho hộp thoại
imageSize "80x80" Nếu imageUrl được thiết lập, bạn có thể đặt [rộng] x [cao] cho hình ảnh

Ví dụ

  1. Thông báo cơ bản 

    swal("Here's a message!")
  2. Thông báo với text phụ bên dưới 

    swal("Here's a message!", "It's pretty, isn't it?")
  3. Thông báo thành công 

    swal("Good job!", "You clicked the button!", "success")
  4. Thông báo tự đóng sau 2 giây 

    swal({
    title: "Auto close alert!",
    text: "I will close in 2 seconds.",
    timer: 2000 
});
  5. Thông báo với nút xác nhận 

    swal({
      title: "Are you sure?",
      text: "You will not be able to recover this imaginary file!",
      type: "warning",
      showCancelButton: true,
      confirmButtonColor: "#DD6B55",
      confirmButtonText: "Yes, delete it!",
      cancelButtonText: "No, cancel plx!",
      closeOnConfirm: false,
      closeOnCancel: false },
      function(isConfirm){
          if (isConfirm) {
              swal("Deleted!", "Your imaginary file has been deleted.", "success");
          } else {
              swal("Cancelled", "Your imaginary file is safe :)", "error");
          }
    });
  6. Thông báo với icon tùy chỉnh 

    swal({
    title: "Sweet!",
    text: "Here's a custom image.",
    imageUrl: "images/icon-sweet.jpg"
});
  7. Xóa comment mẫu

    Nhận ra ưu điểm nổi trội của Sweet alert, chúng tôi cũng đã quết định sử dụng plugin này thay cho hộp thoại alert mặc định của javascript, bạn có thể trải nhiệm thử bằng cách xóa các comment mẫu bên dưới bài viết này
